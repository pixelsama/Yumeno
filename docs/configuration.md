# Configuration

All site choices live in the private overlay's `site/config/` directory.

- `site.json`: identity, text/image brand, author, navigation, hero, about, feature flags, home and footer.
- `scenes.json`: data-driven hero/ambient artwork with attribution, focus, mask, colors, priority, weight and AND-combined conditions.
- `music.json`: local or remote tracks, volume and player availability.

Root-relative media paths are recommended. Yumeno rewrites them for the configured GitHub Pages base path.
