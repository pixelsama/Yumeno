# Bundled fonts

`serif.css` + `serif-sc/*.woff2` — Source Han Serif SC VF（思源宋体可变字体，字重 250–900），
subsets split by unicode-range (common Chinese L1/L2 tiers + ASCII, punctuation, kana and
fullwidth forms from L3), generated from the npm package
[`cn-fontsource-source-han-serif-sc-vf`](https://www.npmjs.com/package/cn-fontsource-source-han-serif-sc-vf)
（中文网字计划）。

Browsers only download the subsets that cover the characters actually used on a page.
